## auto_route tutorial

### Installation

```
dependencies:
  auto_route: ^0.6.7

dev_dependencies:
  auto_route_generator: ^0.6.8
  build_runner:
```

Tip : You can use ```flutter pub run build_runner watch --delete-conflicting-outputs``` command in your terminal to generate the files required or alter generated file on save.
