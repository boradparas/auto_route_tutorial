## auto_route tutorial

### Installation

```
dependencies:
  auto_route: ^0.6.7

dev_dependencies:
  auto_route_generator: ^0.6.8
  build_runner:
```

Use ```flutter pub run build_runner watch --delete-conflicting-outputs``` to alter generated file on save.
